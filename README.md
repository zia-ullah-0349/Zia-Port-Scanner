# ZIA Port Scanner ⚔️🐍

> **Author:** Zia Ullah 🇵🇰
---
## 🎯 Project Goal
A lightweight, educational Python port scanner built with `socket` to understand network reconnaissance. Scans TCP ports 1–100 on a target to identify open doors. Built for learning, not for harming.

**Core Principle:** `Learn to protect, not to kill.` 🛡️

---

## ⚡ Features - v2.0
- **Multi-Port Scanning:** Scans ports 1 to 100 using a `for` loop 🔁
- **Fast Timeout:** 1-second timeout per port for speed ⚡
- **Clean Output:** Only shows `OPEN` ports 🤫
- **Zero Dependencies:** Built with pure Python `socket` 📦
- **Ethical by Design:** Clear legal guidelines included ✅

## 🚀 How to Use
1. **Copy the code** from `port_scanner.py` in this repo
2. **Paste in VS Code** and save as `port_scanner.py` 💾
3. **Run in terminal:** `python port_scanner.py` ⚡
4. **Enter Target IP** when prompted

### ✅ Legal Targets for Practice
| Target | Type | Purpose |
| --- | --- | --- |
| `127.0.0.1` | Localhost | Scan your own machine. 100% safe |
| `192.168.1.1` | Your Router | Audit your home network |
| `scanme.nmap.org` | Public Server | Legal target by Nmap for testing |

## ⚖️ Legal & Ethical Disclaimer
This tool is **strictly for educational use** on systems you own or have explicit, written permission to test.

**Unauthorized port scanning is illegal** under Pakistan's PECA Act and international laws. We are White Hat Hackers. We protect, we don't attack. 🛡️
